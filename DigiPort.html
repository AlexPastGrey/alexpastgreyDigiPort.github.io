Solo Project: First Person Shooter (Coded in Unity/C#)

For this project, I coded and designed every aspect. To highlight one aspect, I designed the UI-Input sync integration such that there is synchronisation between the Interface and the Game Scene - For instance, switching a weapon would cause the Unity Image in the top right corner to change depending on what weapon was switched to.

As well, every weapon has a cooldown period that is depicted by the circle in the top right. This is all handled by a C# script that takes in the current active weapon and contains a function to change the active weapon that is called by a separate handler whenever the weapon is switched. This project is one of my personal highlights as a game developer and I consider it a successful Unity project.
